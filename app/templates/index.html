User Profile Service API

Welcome to the User Profile Service

This service manages user profiles, expertise areas, preferences, and connections for the AI Feedback Platform.

Explore API Documentation

Key Endpoints

GET /api/profiles/me
Get current user's profile
PUT /api/profiles/{id}
Update user profile
GET /api/profiles/search
Search for user profiles
POST /api/profiles/{id}/expertise
Add user expertise area
PUT /api/profiles/{id}/preferences/{category}/{key}
Set user preference
POST /api/profiles/{id}/connections
Request connection with another user
